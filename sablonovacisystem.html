Šablonovací systém je část webové stránky, která se stará o prezentační vrstvu webu (-> vykresluje data do šablon). Příklady můžou být například Latte, Smarty, Twig či Dwoo.

Bootstrap

1. Princip fungování

Bootstrap je CSS framework (sada CSS a JavaScript souborů), který usnadňuje tvorbu webů.

Razí dva principy:

  • 1. Responzivita -> Webová stránka by měla být responzivní, což znamená, že se její obsah bude přizpůsobovat velikosti displaye zařízení tak, aby byla za jakýchkoli okolností přehledná.
  • 2. Mobile first -> Aplikace by měla být vyvíjena nejprve pro nejmenší displaye, následně by se měla uzpůsobit větším displayům.

2. Bootstrap Grid

Bootstrap grid je hlavní předností Bootstrapu (eh, býval, dokud nebyl grid přidán i do základního CSS).

Grid se obvykle tvoří v containeru (div třídy container), a to za pomocí řádků (div třídy row) a sloupců (div třídy col). Bootstrap grid má přesně 12 sloupců, jednotlivé div col tagy mohou zabírat libovolný počet těchto sloupců (přidáním -číslo za třídu col, například div col-6). Počet sloupců, který jednotlivé col tagy zabírají, se může lišit dle velikosti obrazovky (přidáním -velikost za třídu col, například div col-sm-2).

3. Breakpoints

Breakpoints určují velikost displaye, při které se layout webové stránky změní. Defaultně jich má Bootstrap 5, jsou dobře popsané v dokumentaci.

4. Components

Bootstrap nabízí velké množství předem definovaných komponentů. Všechny jsou v dokumentaci (a stejně by si nikdo nezapamatoval, jak se tvoří), takže... google that shit.


FontAwesome, wow.js, chart.js

... jsou knihovny, které usnadňují tvorbu webových stránek.

1. FontAwesome

FontAwesome je knihovna ikon. Pro její použití je třeba ji buď stáhnout, nebo linknout pomocí CDN. Po provedení tohoto kroku bude možné ikony používat. Tyto ikony se do html vkládají pomocí i tagu, který se obvykle kopíruje z webu (-> různé ikony mají různé třídy).

2. wow.js

wow.js je knihovna animací. Po stáhnutí je potřeba ji linknout (CSS a JS). Pro animaci určitého elementu je potřeba mu dát třídu wow, popřípadě specifikovat animaci, například wow bounceInUp.

3. chart.js

char.js je knihovna s grafy. Stejně je to Ctrl+C, Ctrl+V, tak wtf.
